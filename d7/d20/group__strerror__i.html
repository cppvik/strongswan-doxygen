const char* strerror_safe ( int  errnum)

Thread-safe wrapper around strerror and strerror_r.

This is required because the first is not thread-safe (on some platforms) and the second uses two different signatures (POSIX/GNU) and is impractical to use anyway.

Parameters
errnumerror code (i.e. errno)
Returns
error message

Use real strerror() below
