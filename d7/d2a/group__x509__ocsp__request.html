x509_ocsp_request_t* x509_ocsp_request_gen ( certificate_type_t  type,
va_list  args 
)

Generate a X.509 OCSP request.

The resulting builder accepts: BUILD_CA_CERT: CA of the checked certificates, exactly one BUILD_CERT: certificates to check with the request, at least one BUILD_SUBJECT: subject requesting check, optional BUILD_SIGNING_CERT: certificate to create requestor signature, optional BUILD_SIGNING_KEY: private key to create requestor signature, optional

Parameters
typecertificate type, CERT_X509_OCSP_REQUEST only
argsbuilder_part_t argument list
Returns
OCSP request, NULL on failure

See header.
