Helper function to (un-)register IPsec kernel interfaces from plugin features.

This function is a plugin_feature_callback_t and can be used with the PLUGIN_CALLBACK macro to register an IPsec kernel interface constructor.

Parameters

featureassociated plugin feature
regTRUE to register, FALSE to unregister
