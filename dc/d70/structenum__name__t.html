Detailed Description

Struct to store names for enums.

To print the string representation of enumeration values, the strings are stored in these structures. Every enum_name contains a range of strings, multiple ranges are linked together. Use the convenience macros to define these linked ranges.

For a single range, use:

ENUM(name, first, last, string1, string2, ...)

For multiple linked ranges, use:

ENUM_BEGIN(name, first, last, string1, string2, ...)
ENUM_NEXT(name, first, last, last_from_previous, string3, ...)
ENUM_NEXT(name, first, last, last_from_previous, string4, ...)
ENUM_END(name, last_from_previous)

The ENUM and the ENUM_END define a enum_name_t pointer with the name supplied in "name".

Resolving of enum names is done using a printf hook. A printf fromat character N is replaced by the enum string. Printf needs two arguments to resolve a N, the enum_name_t* (the defined name in ENUM_BEGIN) followed by the numerical enum value.
