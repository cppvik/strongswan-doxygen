void(* bus_t::add_logger)(bus_t *this, logger_t *logger)

Register a logger with the bus.

The logger is passive; the thread which emitted the event processes the logger routine. This routine may be called concurrently by multiple threads. Recursive calls are not prevented, so logger that may cause recursive calls are responsible to avoid infinite loops.

During registration get_level() is called for all log groups and the logger is registered to receive log messages for groups for which the requested log level is > LEVEL_SILENT and whose level is lower or equal than the requested level.

To update the registered log levels call add_logger again with the same logger and return the new levels from get_level().

Parameters
loggerlogger to register.

void(* bus_t::set_sa)(bus_t *this, ike_sa_t *ike_sa)

Set the IKE_SA the calling thread is using.

To associate a received log message with an IKE_SA without passing it as parameter each time, the thread registers the currently used IKE_SA during check-out. Before check-in, the thread unregisters the IKE_SA. This IKE_SA is stored per-thread, so each thread has its own IKE_SA registered.

Parameters
ike_saike_sa to register, or NULL to unregister
