status_t(* linked_list_t::find_first)(linked_list_t *this, linked_list_match_t match, void **item,...)

Find the first matching element in the list.

The first object passed to the match function is the current list item, followed by the user supplied data. If the supplied function returns TRUE this function returns SUCCESS, and the current object is returned in the third parameter, otherwise, the next item is checked.

If match is NULL, *item and the current object are compared.

Warning
Only use pointers as user supplied data.
Parameters

itemthe list item, if found

Returns

int(* linked_list_t::remove)(linked_list_t *this, void *item, bool(*compare)(void *, void *))

Remove items from the list matching the given item.

If a compare function is given, it is called for each item, with the first parameter being the current list item and the second parameter being the supplied item. Return TRUE from the compare function to remove the item, return FALSE to keep it in the list.

If compare is NULL, comparison is done by pointers.
