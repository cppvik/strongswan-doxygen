unbound_rr_t* unbound_rr_create_frm_ldns_rr ( ldns_rr *  rr)

Create an unbound_rr instance from a Resource Record given by a ldns_struct_rr from the ldns library.

Returns
Resource Record, NULL on error

The ldns library splits the RDATA field of a RR in various rdf. Here we reassemble these rdf to get the RDATA field of the RR.
