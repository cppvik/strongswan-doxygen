FILE*(* parser_helper_t::file_next)(parser_helper_t *this)

Get the next file to process.

This will return NULL if all files matching the most recent pattern have been handled. If there are other patterns the next call will then return the next file matching the previous pattern.

When hitting <<EOF>> first call

PREFIXpop_buffer_state(yyscanner);

then call this method to check if there are more files to include for the most recent call to file_include(), if so, call

PREFIXset_in(file, helper->scanner);
PREFIXpush_buffer_state(PREFIX_create_buffer(file, YY_BUF_SIZE,
helper->scanner), helper->scanner);

If there are no more files to process check YY_CURRENT_BUFFER and if it is FALSE call yyterminate().

Returns
next file to process, or NULL (see comment)
