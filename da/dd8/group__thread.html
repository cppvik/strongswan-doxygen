void thread_cleanup_popall ( )

Pop and execute all cleanup handlers in reverse order of registration.

This function is for very special purposes only, where the caller exactly knows which cleanup handlers have been pushed. For regular use, a caller should thread_cleanup_pop() exactly the number of handlers it pushed using thread_cleanup_push().
