Detailed Description

A set of DNS Resource Records.

Represents a RRset as defined in RFC 2181. This RRset consists of a set of Resource Records with the same label, class and type but different data.

The DNSSEC signature Resource Records (RRSIGs) which sign the RRs of this set are also part of an object of this type.
