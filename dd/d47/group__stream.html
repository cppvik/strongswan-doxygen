typedef bool(* stream_cb_t)(void *data, stream_t *stream)

Callback function prototype, called when stream is ready.

It is not allowed to destroy the stream nor to call on_read()/on_write/() during the callback.

As select() may return even if a read()/write() would actually block, it is recommended to use the non-blocking calls and handle return values appropriately.

Parameters
datadata passed during callback registration
streamassociated stream
Returns
FALSE unregisters the invoked callback, TRUE keeps it
