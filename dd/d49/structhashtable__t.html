void*(* hashtable_t::get_match)(hashtable_t *this, const void *key, hashtable_equals_t match)

Returns the value with a matching key, if the hash table contains such an entry, otherwise NULL is returned.

Compared to get() the given match function is used to compare the keys for equality. The hash function does have to be deviced properly in order to make this work if the match function compares keys differently than the equals function provided to the constructor. This basically allows to enumerate all entries with the same hash value.

Parameters
keythe key to match against
matchmatch function to be used when comparing keys
Returns
