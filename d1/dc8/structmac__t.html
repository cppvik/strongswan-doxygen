bool(* mac_t::get_mac)(mac_t *this, chunk_t data, u_int8_t *out) __attribute__((warn_unused_result))

Generate message authentication code.

If out is NULL, no result is given back. A next call will append the data to already supplied data. If out is not NULL, the mac of all apended data is calculated, written to out and the internal state is reset.

Parameters
datachunk of data to authenticate
outpointer where the generated bytes will be written
Returns
TRUE if mac generated successfully
