stream_t* stream_create_unix ( char *  uri)

Create a stream for UNIX sockets.

UNIX URIs start with unix://, followed by the socket path. For absolute paths, an URI looks something like:

unix:///path/to/socket

Parameters
uriUNIX socket specific URI, must start with "unix://"
Returns
stream instance, NULL on failure
