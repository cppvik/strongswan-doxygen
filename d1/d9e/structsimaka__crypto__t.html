bool(* simaka_crypto_t::derive_keys_full)(simaka_crypto_t *this, identification_t *id, chunk_t data, chunk_t *mk, chunk_t *msk)

Derive keys after full authentication.

This methods derives the k_encr/k_auth keys and loads them into the internal crypter/signer instances. The passed data is method specific: For EAP-SIM, it is "n*Kc|NONCE_MT|Version List|Selected Version", for EAP-AKA it is "IK|CK".

Parameters
idpeer identity
datamethod specific data
mkchunk receiving allocated master key MK
mskchunk receiving allocated MSK
Returns
TRUE if keys allocated and derived successfully
