Enumeration Type Documentation

anonymous enum

Message type.

The client can send a batch of request messages, containing DUMP, LOOKUP or REGISTER_* messages. The server immediately starts sending responses for these messages, using ENTRY or NOTIFY_* messages. A client MUST send an END message to complete a batch. The server will send any remaining responses, but will not accept new requests and closes the connection when complete.
