id_match_t(* identification_t::matches)(identification_t *this, identification_t *other)

Check if an ID matches a wildcard ID.

An identification_t may contain wildcards, such as *.strongswan.org. This call checks if a given ID (e.g. tester.strongswan.org) belongs to a such wildcard ID. Returns > 0 if

  • IDs are identical
  • other is of type ID_ANY
  • other contains a wildcard and matches this

The larger the return value is, the better is the match. Zero means no match at all, 1 means a bad match, and 2 a slightly better match.

Parameters
otherthe ID containing one or more wildcards
Returns
match value as described above
