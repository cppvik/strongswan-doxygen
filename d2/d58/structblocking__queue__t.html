void(* blocking_queue_t::destroy_function)(blocking_queue_t *this, void(*)(void *))

Destroys a queue and its objects using a cleanup function.

If a queue and its contents should get destroyed using a specific cleanup function, use destroy_function. This is useful when the list contains malloc()-ed blocks which should get freed, e.g.: queue->destroy_function(queue, free);

Note
No thread must wait in dequeue() when this function is called
Parameters
functionfunction to call on each object

void(* blocking_queue_t::destroy_offset)(blocking_queue_t *this, size_t offset)

Destroys a queue and its objects using the given destructor.

If a queue and the contained objects should be destroyed, use destroy_offset. The supplied offset specifies the destructor to call on each object. The offset may be calculated using the offsetof macro, e.g.: queue->destroy_offset(queue, offsetof(object_t, destroy));

Note
No thread must wait in dequeue() when this function is called
Parameters
offsetoffset of the objects destructor
