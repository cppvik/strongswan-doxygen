PB-TNC State Machine (see section 3.2 of RFC 5793) 

       Receive CRETRY        SRETRY
            or SRETRY   +----------------+
                 +--+   |                |
                 v  |   v                |
                +---------+  CRETRY  +---------+
      CDATA     | Server  |<---------| Decided | CLOSE
   +----------->| Working |--------->|         |-------+
   |            +---------+  RESULT  +---------+       |
   |                ^ |  |                             v
   |                | |  +---------------------->=======
 ========           | |              CLOSE       " End "
 " Init "      CDATA| |SDATA                     =======
 ========           | |                          ^    ^
   |  |             | v                          |    |
   |  | SDATA   +---------+          CLOSE       |    |
   |  +-------->| Client  |----------------------+    |
   |            | Working |                           |
   |            +---------+                           |
   |                |  ^                              |
   |                +--+                              |
   |            Receive CRETRY                        |
   |   CLOSE                                          |
   +--------------------------------------------------+
