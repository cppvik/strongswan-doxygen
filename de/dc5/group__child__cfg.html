Create a configuration template for CHILD_SA setup.

The "name" string gets cloned.

The lifetime_cfg_t object gets cloned. To prevent two peers to start rekeying at the same time, a jitter may be specified. Rekeying of an SA starts at (x.rekey - random(0, x.jitter)).

After a call to create, a reference is obtained (refcount = 1).

Parameters

lifetimelifetime_cfg_t for this child_cfg
updownupdown script to execute on up/down event
hostaccessTRUE to allow access to the local host
modemode to propose for CHILD_SA, transport, tunnel or BEET

dpd_actionDPD action

inactivityinactivity timeout in s before closing a CHILD_SA
reqidspecific reqid to use for CHILD_SA, 0 for auto assign

mark_outoptional outbound mark (can be NULL)
tfcTFC padding size, 0 to disable, -1 to pad to PMTU
