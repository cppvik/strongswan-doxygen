enum diffie_hellman_group_t

Diffie-Hellman group.

The modulus (or group) to use for a Diffie-Hellman calculation. See IKEv2 RFC 3.3.2 and RFC 3526.

ECP groups are defined in RFC 4753 and RFC 5114. ECC Brainpool groups are defined in RFC 6954.
