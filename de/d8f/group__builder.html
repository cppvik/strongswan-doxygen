Typedef Documentation

typedef void*(* builder_function_t)(int subtype, va_list args)

Constructor function to build credentials.

Any added parts are cloned/refcounted by the builder implementation. Callers may need to free the passed resources themselves.

Parameters
subtypeconstructor specific subtype, e.g. a certificate_type_t
argslist of builder part types, followed by parts, BUILD_END
Returns
builder specific credential, NULL on error
