bool(* stream_t::read_all)(stream_t *this, void *buf, size_t len)

Read data from the stream, avoiding short reads.

This call is always blocking, and reads until len has been read completely. If the connection is closed before enough bytes could be returned, errno is set to ECONNRESET.
