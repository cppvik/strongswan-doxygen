bool cred_encoding_args ( va_list  args,
  ... 
)

Helper function for cred_encoder_t implementations to parse argument list.

Credential encoder functions get a variable argument list to parse. To simplify the job, this function reads the arguments and returns chunks for each part. The argument list of this function takes a cred_encoding_part_t, followed by a data pointer receiving the value, terminated by CRED_PART_END.

Parameters
argsargument list passed to credential encoder function
...list of (cred_encoding_part_t, data*)
Returns
TRUE if all parts found, FALSE otherwise
