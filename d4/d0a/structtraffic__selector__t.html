Detailed Description

Object representing a traffic selector entry.

A traffic selector defines an range of addresses and a range of ports.

If the protocol is ICMP or ICMPv6 the ICMP type and code are stored in the port field as follows: The message type is placed in the most significant 8 bits and the code in the least significant 8 bits. Utility functions are provided to extract the individual values.
