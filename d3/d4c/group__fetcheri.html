Enumeration Type Documentation

enum fetcher_option_t

Fetching options to use for fetcher_t.fetch() call.

Enumerator
FETCH_REQUEST_DATA 

Data to include in fetch request, e.g. on a HTTP post. Additional argument is a chunk_t

FETCH_REQUEST_TYPE 

Mime-Type of data included in FETCH_REQUEST_DATA. Additional argument is a char*.

FETCH_REQUEST_HEADER 

HTTP header to be sent with with the fetch request. Additional argument is a char*.

FETCH_HTTP_VERSION_1_0 

Use HTTP Version 1.0 instead of 1.1. No additional argument is needed.

FETCH_TIMEOUT 

Timeout to use for fetch, in seconds. Additional argument is u_int

FETCH_RESPONSE_CODE 

Sets a pointer to a variable that receives the request's response code. Additional argument is a u_int*.

FETCH_CALLBACK 

Callback to invoke with each chunk of data. Additional argument fetch_callback_t. If this option is not given, the fetcher_default_callback is used, which accumulates the data into an allocated chunk.

FETCH_SOURCEIP 

Source IP address to bind for a fetch. Additional argument is a host_t*, which may be NULL.

FETCH_END 

end of fetching options
