bool library_init ( char *  settings,
const char *  namespace 
)

Initialize library, creates "lib" instance.

library_init() may be called multiple times in a single process, but each caller must call library_deinit() for each call to library_init().

The settings and namespace arguments are only used on the first call.

Parameters
settingsfile to read settings from, may be NULL for default
namespacename of the binary that uses the library, determines the first section name when reading config options. Defaults to libstrongswan if NULL.
Returns
FALSE if integrity check failed
