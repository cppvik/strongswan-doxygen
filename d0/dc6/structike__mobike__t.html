Detailed Description

Task of type ike_mobike, detects and handles MOBIKE extension.

The MOBIKE extension is defined in RFC4555. It allows to update IKE and IPsec tunnel addresses. This tasks handles the MOBIKE_SUPPORTED notify exchange to detect MOBIKE support, allows the exchange of ADDITIONAL_*_ADDRESS to exchange additional endpoints and handles the UPDATE_SA_ADDRESS notify to finally update endpoints.
