)
static

Check if we already have created an IKE_SA based on the initial IKE message with the given hash. If not the hash is stored, the hash data is not(!) cloned.

Also, the local SPI is returned. In case of a retransmit this is already stored together with the hash, otherwise it is newly allocated and should be used to create the IKE_SA.

Returns
ALREADY_DONE if the message with the given hash has been seen before NOT_FOUND if the message hash was not found FAILED if the SPI allocation failed
