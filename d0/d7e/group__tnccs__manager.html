bool tnccs_method_register ( plugin_t *  plugin,
plugin_feature_t *  feature,
bool  reg,
void *  data 
)

Helper function to (un-)register TNCCS methods from plugin features.

This function is a plugin_feature_callback_t and can be used with the PLUGIN_CALLBACK macro to register a TNCCS method constructor.

Parameters
pluginplugin registering the TNCCS method constructor
featureassociated plugin feature
regTRUE to register, FALSE to unregister.
datadata passed to callback, a tnccs_constructor_t
