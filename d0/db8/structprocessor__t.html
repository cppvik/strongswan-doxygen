void(* processor_t::set_threads)(processor_t *this, u_int count)

Set the number of threads to use in the processor.

If the number of threads is smaller than number of currently running threads, thread count is decreased. Use 0 to disable the processor.

This call does not block and wait for threads to terminate if the number of threads is reduced. Instead use cancel() for that during shutdown.
