enum payload_type_t

Payload-Types of an IKE message.

Header and substructures are also defined as payload types with values from PRIVATE USE space.

Enumerator
PL_NONE 

End of payload list in next_payload

PLV1_SECURITY_ASSOCIATION 

The security association (SA) payload containing proposals.

PLV1_PROPOSAL 

The proposal payload, containing transforms.

PLV1_TRANSFORM 

The transform payload.

ID payload.

Create an empty payload.

Useful for the parser, who wants a generic constructor for all payloads. It supports all payload_t methods. If a payload type is not known, an unknwon_paylod is created with the chunk of data in it.

Parameters
