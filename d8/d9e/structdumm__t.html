bool(* dumm_t::add_overlay)(dumm_t *this, char *dir)

Add an overlay to all guests.

Directories named after the guests are created, if they do not exist in the given overlay directory.

If adding the overlay on at lest one guest fails, FALSE is returned and the overlay is again removed from all guests.

bool(* dumm_t::load_template)(dumm_t *this, char *name)

Loads a template, create a new one if it does not exist.

This is basically a wrapper around add/del_overlay to simplify working with overlays. Templates are located in a predefined directory, so that only a name for the template has to be specified here. Only one template can be loaded at any one time (but other overlays can be added on top or below a template).

Parameters
namename of the template to load, NULL to unload
Returns
FALSE if load/create failed
