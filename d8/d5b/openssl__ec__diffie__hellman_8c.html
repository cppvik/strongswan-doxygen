static bool compute_shared_key ( private_openssl_ec_diffie_hellman_t *  this,
chunk_t *  shared_secret 
)
static

Compute the shared secret.

We cannot use the function ECDH_compute_key() because that returns only the x coordinate of the shared secret point (which is defined, for instance, in 'NIST SP 800-56A'). However, we need both coordinates as RFC 4753 says: "The Diffie-Hellman public value is obtained by concatenating the x and y values. The format of the Diffie-Hellman shared secret value is the same as that of the Diffie-Hellman public value."
