stream_t* stream_create_tcp ( char *  uri)

Create a stream for TCP sockets.

TCP URIs start with tcp://, followed by a hostname (FQDN or IP), followed by a colon separated port. A full TCP uri looks something like:

tcp://srv.example.com:5555 tcp://0.0.0.0:1234 tcp://[fec2::1]:7654

There is no default port, so a colon after tcp:// is mandatory.

Parameters
uriTCP socket specific URI, must start with "tcp://"
Returns
stream instance, NULL on failure
