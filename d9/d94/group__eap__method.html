typedef eap_method_t*(* eap_constructor_t)(identification_t *server, identification_t *peer)

Constructor definition for a pluggable EAP method.

Each EAP module must define a constructor function which will return an initialized object with the methods defined in eap_method_t. Constructors for server and peers are identical, to support both roles of a EAP method, a plugin needs register two constructors in the eap_manager_t. The passed identites are of type ID_EAP and valid only during the constructor invocation.

Parameters
serverID of the server to use for credential lookup
peerID of the peer to use for credential lookup
Returns
implementation of the eap_method_t interface
