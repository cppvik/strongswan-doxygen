char* path_basename ( const char *  path)

Like basename(3) returns the filename part of the given null-terminated path, i.e. the part following the final '/' (or '.' if path is empty or NULL). Trailing '/' are not counted as part of the pathname.

The difference is that it does this in a thread-safe manner (i.e. it does not use static buffers) and does not modify the original path.

char* path_dirname ( const char *  path)

Like dirname(3) returns the directory part of the given null-terminated pathname, up to but not including the final '/' (or '.' if no '/' is found). Trailing '/' are not counted as part of the pathname.

The difference is that it does this in a thread-safe manner (i.e. it does not use static buffers) and does not modify the original path.
