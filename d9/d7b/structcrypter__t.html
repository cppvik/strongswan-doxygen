The length of the iv must equal to get_iv_size(), while the length of data must be a multiple of get_block_size(). If decrpyted is NULL, the encryption is done in-place (overwriting data).

Get the block size of the crypto algorithm.

get_block_size() returns the smallest block the crypter can handle, not the block size of the underlying crypto algorithm. For counter mode, it is usually 1.
